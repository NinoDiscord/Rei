## Installation
### Prerequisites
Before running Rei, you must install **Go 1.16** or higher to use.

### Building
```sh
$ git clone https://github.com/NinoDiscord/Rei.git && cd Rei
$ go get && go build
```

You should get a `rei` binary if on UNIX or `rei.exe` on Windows.
